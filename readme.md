# **Prerequisites**

- [Git](https://git-scm.com)
- [Node.js](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/getting-started)
- [Docker](https://www.docker.com/get-started)

# **Deploy**

```
yarn install
cp .example.env ./.dev.env
```

# **Migrations**

```
yarn migration:run
```

# **Localhost**

```
yarn start:dev
```

# **Tests**

```
yarn test
yarn test:ci
```
